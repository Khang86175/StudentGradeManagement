# Hệ thống Quản lý Điểm THPT (QLDS_OOP_Project)

Đây là dự án cho môn học Lập trình Hướng đối tượng.

## 1. Mô tả
Ứng dụng desktop xây dựng bằng JavaFX và SQLite để quản lý thông tin học sinh, giáo viên, lớp học, môn học và điểm số.

## 2. Công nghệ sử dụng
- Ngôn ngữ: Java 17
- Framework UI: JavaFX
- CSDL: SQLite
- Quản lý phiên bản: Git & GitHub

## 4. Kiến trúc
Dự án áp dụng kiến trúc Model-View-Controller (MVC) kết hợp với Data Access Object (DAO) Pattern.
